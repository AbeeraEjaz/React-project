# 🎓 Student Portal

A complete React-based student management system with CRUD operations.

## ✨ Features

- ➕ Add students
- 📋 View student list  
- ✏️ Edit student information
- 🗑️ Delete students
- 🔍 Search students
- 📱 Responsive design
- 🌐 Multiple pages (React Router)

## 🛠️ Technologies Used

- React 18
- CSS3
- React Router DOM
- Vite

## 🚀 How to Run

```bash
npm install
npm run dev
